# Venture Day Schedule

Briana runs a venture-day rhythm. Each weekday is anchored to a venture so deep
work blocks don't fragment. Use this to weight priorities in daily briefings.

| Day       | Focus                         |
|-----------|-------------------------------|
| Monday    | The Trades Show (TTS)         |
| Tuesday   | Fractal / Aura (client work)  |
| Wednesday | The Corral + Detto            |
| Thursday  | Catch-up / admin / deep work  |
| Friday    | Filming / content production  |
| Weekend   | Rest, reflection, optional planning |

Tasks scheduled on a day that mismatches their venture are not errors — flag
them softly ("This is a TTS task on a Corral day — worth moving?") but never
override Briana's explicit scheduling.
